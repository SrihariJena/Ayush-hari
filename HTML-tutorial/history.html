History of HTML:

It was created by "Tim Berners-Lee" in late 1991 but wasn't officially released. It was published in 1995 as HTML 2.0. HTML 4.01 was published in late 1999 and was a major version of HTML. HTML 5 can be said for an extended version of HTML 4.01 which was published in the year 2012.

Features of HTML:

  1. It is very simple and easy language.
  2. It can be easily understood and modified.
  3. It is very easy to make one effective presentation with HTML, because it has a lot of formatting tags.
  4. It is a markup language so it provides a flexible way to design web pages along within the text.
  5. It facilitates programmers to add a link on the wb pages (By HTML Anchor Tag). So, it enhances the interest of browsing of the user.
  6. It is platform independent because it can be displayed on any platform like window, linux etc.
  7. It facilitates the prorammer to add graphic videos and sound to the web pages which makes it more effective and interactive.
  8. HTML is a case-sensitive language which means we can use tags either in lowe case or upper case.
