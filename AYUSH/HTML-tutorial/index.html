HTML Introduction

HTML stands for HyperText Markup Language. It is used for creating web pages. It consists of a series of elements. This element tells the browser how to display the contents. This element label pieces of content such as this is a heading, this is a paragraph, this is a link.

Why HyperText Markup Language:

Hyperlink:

It simply means "text within text". A text as a link within it, is a hypertext. Whenever we click on a link which brings us to a new webpage. We have clicked on a hypertext. It is a way to link two or more webpages (HTML Documents).

Markup Language:

It is a computer language i.e., used to apply layout and formatting convertion to a text document. It makes text more interactive and dynamic. It can turn text into images, tables, links, etc.

Website:

It is a collection of webpages. It is divided into two types:

  1. Static website: Ita contains webpages with fixed contents. Each page is coded in HTML and displayed the same information to every viewers.
  2. Dynamic website: It contains webpages that are generated in real-time.
Static Website Dynamic Website
The content and layout of webpage is fixed. The content and layout may change during run-time.
It never use database. Database is used to generate dynamic content through queries.
It directly run on the browser and don't require any server-side application program. It runs on the server site application programs and display the result.
Static pages are easy to develop. Dynamic pages development requires programming skill.

Web Browser:

It's purpose (chrome, firefox, opera, edge, safari) is to read HTML document and display them correctly. A browser doesn't display the HTML tags but uses them to determine how to display the document.
