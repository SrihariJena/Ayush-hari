How to create 1st Web page:

To create 1st webpage follow the steps:

  1. Open Notepad.
  2. Write a HTML Code as
<!DOCTYPE html> <html> <head></head> <body> Hello World! </body> </html>
  1. Save the file on the computer with extension .html or .htm.
  2. Open the saved file in browser(chrome) and double click or right click and open with the browser.
    The output will be shown as:
Hello World!

HTML tags:

This tags are like keywords which defines that how web browser will format and display the content with the help of tags. A web browser can distinguish between an HTML content and a simple content. HTML tags contain three parts:

  • Opening tags
  • Closing tag
  • But Some HTML tags are unclosed tags. ex: content...

    • HTML Document:

    An HTML Documents must starts with a document type declaration and ends with </html>.

    <!DOCTYPE html> <html> <head <!-- content --> </head> <body> <!-- content --> </body> </html>

    The visible part of the document is between <body> and </body>.

    HTML Description:

    • The <html> element is the root element of html page.
    • The <head>element contains meta information about the html page.
    • The <title> element specifies a title for the html page, which is shown in the browser's title bar or in the page tab.
    • <body>element defines the document body and is a container for all the visible contents such as heading, paragraphs, images, hyperlinks, tables, lists, etc.
