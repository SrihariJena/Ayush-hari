External CSS

External CSS is written in a separate file with a .css extension and linked to the HTML file using the <link> tag.

Example:

HTML:
<!-- elements and content -->
<link rel="stylesheet" href="styles.css">

CSS (styles.css):
h1 {
color: blue;
}

This paragraph will be styled by an external CSS file (if connected).

Internal CSS

Internal CSS is written within a <style> tag in the <head> section of the HTML document.

Example:

HTML:
<!-- elements and content -->
<style>
p {
color: green;
font-weight: bold;
}
</style>
<p>This paragraph is styled with internal CSS.</p>

This paragraph is styled with internal CSS.

Inline CSS

Inline CSS is written directly within an HTML element's using style attribute.

Example:

HTML:

<p style="color: red; font-size: 20px;">This paragraph is styled with inline CSS.</p>

This paragraph is styled with inline CSS.

Priority Rules

If all three are applied to the same element, the order of precedence is:

  • Inline CSS overrides Internal and External.
  • Internal CSS overrides External CSS.
  • External CSS has the lowest priority.

Specificity of Selectors

Beyond the type of CSS (inline, internal, external), the specificity of selectors also plays a role:

  • Inline styles (style="...") have the highest specificity.
  • IDs (#id) are more specific than classes (.class) or tags (div, p).
  • Classes (.class) are more specific than element tags (div, p).
  • More specific selectors override less specific ones.
