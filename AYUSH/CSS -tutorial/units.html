Absolute Units

Fixed-size units, unaffected by parent or screen size.

css:
div {
width: 200px;
height: 2in;
padding: 5mm;
}

Relative Units

Flexible units, depending on the parent element or viewport.

css:
div {
width: 50%;
font-size: 1.2em;
margin: 5vw;
}

Relative Units: %, rem, em

These units are flexible and adjust based on the parent element or root element.

css:
/* Percentage Example */
div {
width: 50%; /* 50% of its parent's width */
height: 100%; /* 100% of its parent's height */
}
/* rem Example */
body {
font-size: 16px; /* Base font size */
}
h1 {
font-size: 2rem; /* 2 x 16px = 32px */
}
p {
font-size: 1rem; /* Equal to 16px */
}
/* em Example */
.container {
font-size: 20px; /* Base font size for this container */
}
.container h2 {
font-size: 1.5em; /* 1.5 x 20px = 30px */
}
.container p {
font-size: 0.8em; /* 0.8 x 20px = 16px */
}

Time Units

Used in animations or transitions.

css:
button {
transition: all 0.3s;
}

Angle Units

Used for rotations or gradients.

css:
div {
transform: rotate(45deg);
}

Example with HTML

<!DOCTYPE html>
<html>
<head>
<style>
.box {
width: 80%;
height: 200px;
margin: 5vw;
font-size: 1.5em;
transform: rotate(45deg);
transition: all 0.5s;
}
</style>
</head>
<body>
<div class="box">CSS Units Example</div>
</body>
</html>
