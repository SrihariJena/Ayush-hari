Text Properties in CSS

text-align

Specifies the horizontal alignment of text.
Values: left, right, center, justify

text-align: center;

text-decoration

Specifies the decoration added to text.
Values: none, underline, overline, line-through

text-decoration: underline;

text-transform

Controls the capitalization of text.
Values: none, capitalize, uppercase, lowercase

css: text-transform: uppercase;

text-indent

Specifies the indentation of the first line in a text block.
Any length value like 20px, 10%, 5em

text-indent: 50px;

letter-spacing

Sets the spacing between characters.
Any length value like 2px, 0.1em, -1px

css: letter-spacing: 2px;

word-spacing

Sets the spacing between words.
Any length value like 10px, 5em, 3rem

css: word-spacing: 10px;

line-height

Specifies the space between lines of text.
Numbers (like 1.5), percentages (150%), or lengths (20px)

css: line-height: 1.5;

color

Specifies the color of text.
Values: Named colors (red, blue), hex codes (#000), or RGB (rgb(0,0,255))

css: color: blue;

Example with HTML

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
text-align: center;
text-decoration: underline;
}
p {
text-indent: 20px;
line-height: 1.6;
color: gray;
}
</style>
</head>
<body>
<h1>CSS Text Properties</h1>
<p>This is an example demonstrating text properties in CSS.</p>
</body>
</html>
