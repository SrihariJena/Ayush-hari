Colors in CSS

Colors in CSS can be defined using different methods:

  • Named Colors: Use predefined color names like red, blue, or green.
  • Hexadecimal: Use a #RRGGBB or #RGB format.
  • RGB: Define colors using rgb(red, green, blue).
  • HSL: Define colors using hsl(hue, saturation, lightness).

Examples:

<p class="color-name">This text is red (named color).</p>
<p> class="color-hex">This text is blue (hex color: #007BFF).</p>
<p> class="color-rgb">This text is green (RGB color: rgb(34, 139, 34)).</p>
<p> class="color-hsl">This text is blue (HSL color: hsl(240, 100%, 50%)).</p>
