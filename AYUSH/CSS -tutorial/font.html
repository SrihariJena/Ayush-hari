font-family

Specifies the font(s) to be used. Always include a fallback font or generic family.

css: font-family: 'Arial', 'Helvetica', sans-serif;

font-size

Sets the size of the font
Values: px, em, rem, %, vw, vh, or keywords like small, large.

css: font-size: 16px;

font-style

Defines the style of the font.
Values: normal, italic, oblique.

css: font-style: italic;

font-style

Defines the style of the font.
Values: normal, italic, oblique.

css: font-style: italic;

font-variant

Specifies small-caps or other font variations.
Values: normal, small-caps.

css: font-variant: small-caps;

font-weight

Specifies the weight (thickness) of the font.
Values: normal, bold, lighter, bolder, or numeric values (100 to 900).

css: font-weight: bold;

line-height

Sets the height of a line of text.
Values: Numeric (e.g., 1.5), length units (e.g., 20px), or percentages.

css: line-height: 1.5;

Font-Shorthand

p { font: italic small-caps bold 16px/1.5 'Arial', sans-serif; }

This shorthand sets:

  1. Font style: italic
  2. Font variant: small-caps
  3. Font weight: bold
  4. Font size: 16px
  5. Line height: 1.5
  6. Font family: 'Arial', sans-serif

Example with HTML

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Font Property Example</title>
<style>
body {
font-family: 'Georgia', serif;
font-size: 18px;
line-height: 1.6;
}
h1 {
font: bold 2em/1.2 'Arial', sans-serif;
}
p {
font: italic small-caps normal 1.2em/1.5 'Times New Roman', serif;
}
</style>
</head>
<body>
<h1>CSS Font Property</h1>
<p>This is an example paragraph demonstrating font properties in CSS. </p>
</body>
</html>
