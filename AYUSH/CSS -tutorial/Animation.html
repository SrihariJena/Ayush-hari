CSS Animations

CSS animations allow you to animate HTML elements without relying on JavaScript. They can change properties like size, position, and color over time using keyframes.

Key Properties of CSS Animations

  • animation-name: Specifies the name of the keyframes.
  • animation-duration: Defines how long the animation lasts (e.g., 3s, 2.5s).
  • animation-timing-function: Determines the pacing of the animation (e.g., ease, linear).
  • animation-delay: Specifies when the animation should start after the page loads.
  • animation-iteration-count: Sets how many times the animation repeats (e.g., infinite, 2).
  • animation-direction: Specifies whether the animation runs forward, backward, or alternates (normal, reverse, alternate).

Examples of CSS Animations

1. Fade In

This animation gradually changes the opacity of the element from 0 to 1, creating a fade-in effect.

@keyframes fadeIn {
    0% { opacity: 0; }
    100% { opacity: 1; }
}

.example {
    animation: fadeIn 3s ease-in-out infinite;
}

2. Slide Left

This animation slides the element horizontally from right to left and then back.

@keyframes slideLeft {
    0% { transform: translateX(100%); }
    50% { transform: translateX(0); }
    100% { transform: translateX(-100%); }
}

.example {
    animation: slideLeft 2s ease-in-out infinite;
}

3. Rotate

This animation rotates the element continuously in a clockwise direction.

@keyframes rotate {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

.example {
    animation: rotate 2s linear infinite;
}
