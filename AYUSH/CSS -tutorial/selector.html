Basic Selectors

Basic selectors are used to target HTML elements directly.

  • * - Universal selector (targets all elements).
  • tagname - Selects all elements of the specified tag (e.g., p, div).
  • .classname - Selects all elements with the specified class.
  • #id - Selects an element with the specified ID.

Example:

HTML:
<div class="example-class" id="example-id">Hello, World! </div>
CSS:
* {
margin: 0; /* Universal selector */
}
div {
color: blue; /* Tag selector */
}
.example-class {
font-weight: bold; /* Class selector */
}
#example-id {
font-size: 20px; /* ID selector */
}

Combinators

Combinators define relationships between selectors.

  • A B - Descendant selector (selects B elements inside A).
  • A > B - Child selector (selects direct children B of A).
  • A + B - Adjacent sibling selector (selects B immediately after A).
  • A ~ B - General sibling selector (selects all B siblings of A).

Example:

HTML:
<div class="container">
<p>Paragraph 1</p>
<p>Paragraph 2</p>
<span>Span inside div</span>
</div>
CSS:
.container p {
color: green; /* Descendant selector */
}
.container > p:first-child {
font-weight: bold; /* Child selector */
}
.container > p:nth-child(2) {
font-weight: 500; /* Child selector */
}
p + span {
color: red; /* Adjacent sibling selector */
}
p ~ span {
background-color: yellow; /* General sibling selector */
}

Pseudo-classes

Pseudo-classes target elements based on their state or position.

  • :hover - Selects an element when hovered.
  • :nth-child(n) - Selects the nth child of a parent element.
  • :focus - Selects an element when focused.
  • :not(selector) - Selects all elements not matching the selector.

Example:

HTML:
<input type="text" placeholder="Focus me!">
<ul>
<li> First item</li>
<li> Second item</li>
<li> Third item</li>
</ul>
CSS:
input:focus {
border: 2px solid blue; /* Focused input */
} li:nth-child(2) {
color: orange; /* Second child in list */
} li:not(:nth-child(2)) {
color: gray; /* All except the second child */
}

Pseudo-elements

Pseudo-elements style specific parts of an element.

  • ::before - Inserts content before the element.
  • ::after - Inserts content after the element.
  • ::first-line - Styles the first line of text within an element.
  • ::first-letter - Styles the first letter of an element.

Example:

HTML:
<p>Welcome to CSS!<p>
CSS:
p::before {
content: "★ ";
color: gold; /* Star before text */
}
p::after {
content: " ★";
color: gold; /* Star after text */
}
p::first-line {
font-weight: bold; /* Bold first line */
}
p::first-letter {
font-size: 2em; /* Enlarged first letter */
}
